🧹 Data Cleaning Project

This project focuses on cleaning and transforming a raw product dataset to make it analysis-ready using Excel / Power Query techniques.

📌 Project Overview

In this project, I performed several data cleaning and transformation steps to improve data quality, consistency, and usability for analysis.

🛠️ Data Cleaning Steps Performed
Filled missing values in Price and Category columns
Corrected spelling errors in Category and Product Name
Removed duplicate records from the dataset
Split the Product ID column into:
Manufacturing Date
Country Code
Merged Product Name and Brand Name to create a new column:
Product Brand
Converted Price column into proper currency format
Standardized Manufacturing Date format to: DD-MM-YY
Applied conditional formatting:
Data bars for Price column to visualize value distribution
Highlighted Electronics category for better visibility

📊 Tools Used
Microsoft Excel / Power Query
Data Cleaning Functions
Conditional Formatting Tools
🎯 Outcome

The dataset is now:

Clean and consistent
Free from duplicates and errors
Structured for analysis and visualization
Easier to interpret for business insights
